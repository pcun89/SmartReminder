# SmartReminder

# Full-Stack Calendar & To-Do Reminder Application

# SmartReminder is a full-stack web application that allows users to manage calendar activities and to-do tasks with priority-based visual indicators. Tasks are color-coded to clearly represent urgency, helping users stay organized and focused.

# 🚀 Features

# 📆 Add activities tied to calendar dates

# ✅ Manage a to-do list

# 🎨 Priority-based color coding

# 🔴 Red — Urgent

# 🔵 Blue — Medium

# 🟢 Green — Low

# 🔄 Real-time updates via REST API

# 🧩 Clean frontend/backend separation
